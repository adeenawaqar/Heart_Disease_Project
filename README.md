# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a patient is likely to have heart disease based on medical information. It is developed using Machine Learning classification algorithms and deployed with Streamlit.

## 🎯 Objective

To predict the possibility of heart disease using patient medical data such as age, blood pressure, cholesterol level, chest pain type, and other clinical features.

## 📂 Dataset

* Dataset: Heart Disease Prediction Dataset
* Type: Structured Medical Dataset
* Target Variable: Heart Disease

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* XGBoost
* Streamlit
* Joblib

## 🤖 Machine Learning Models

The following classification algorithms were implemented:

* Logistic Regression
* Support Vector Machine (SVM)
* Random Forest
* XGBoost

## 📊 Project Workflow

1. Import Libraries
2. Load Dataset
3. Exploratory Data Analysis (EDA)
4. Data Cleaning
5. Handle Missing Values
6. Remove Duplicate Records
7. Encode Categorical Features
8. Train-Test Split
9. Feature Scaling
10. Train Multiple Machine Learning Models
11. Evaluate Models
12. Compare Accuracy
13. Save Best Model
14. Deploy using Streamlit

## 📈 Evaluation Metrics

## ▶️ How to Run

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
